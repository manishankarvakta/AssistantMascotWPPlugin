A WordPress plugin that displays a customizable red square with "Plugin active" text on the frontend and provides a comprehensive admin area for configuration.

== Description ==

Assistant Mascot is a lightweight WordPress plugin that adds a visual indicator to your website showing that the plugin is active. The plugin displays a customizable red square with "Plugin active" text that can be positioned in different corners of your website.

**Key Features:**

* **Visual Indicator**: Displays a red square with "Plugin active" text on your website
* **Customizable Position**: Choose from 4 different positions (top-left, top-right, bottom-left, bottom-right)
* **Color Customization**: Customize both text and background colors
* **Admin Interface**: Easy-to-use settings page in WordPress admin
* **Responsive Design**: Works perfectly on all devices and screen sizes
* **Accessibility**: Keyboard navigation and screen reader support
* **Performance Optimized**: Lightweight and fast loading

**Perfect for:**
* Developers who want to show plugin status
* Website owners who need visual confirmation of active plugins
* Anyone who wants to add a professional touch to their website

== Installation ==

1. Upload the `assistant-mascot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Assistant Mascot' in your admin menu to configure settings
4. Customize the appearance and position of your mascot

== Frequently Asked Questions ==

= What does this plugin do? =

This plugin displays a red square with "Plugin active" text on your website to show that the plugin is working.

= Can I change the colors? =

Yes! You can customize both the text color and background color through the admin settings.

= Where can I position the mascot? =

You can choose from 4 positions: top-left, top-right, bottom-left, or bottom-right.

= Is this plugin mobile-friendly? =

Absolutely! The plugin is fully responsive and works perfectly on all devices.

= Can I disable the mascot? =

Yes, you can easily enable/disable the mascot through the admin settings.

= Does this plugin affect my website's performance? =

No, this plugin is very lightweight and won't impact your website's performance.
